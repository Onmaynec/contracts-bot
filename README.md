# Contracts Bot v1.3.0

Discord бот для управления контрактами и онлайном пользователей.

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Создайте файл `.env` на основе `.env.example` и заполните токен бота
4. Зарегистрируйте команды: `npm run deploy`
5. Запустите бота: `npm start`

## Команды

### Общие команды (доступны всем)

| Команда | Описание |
|---------|----------|
| `/settingsinfo` | Показать все настройки бота на сервере |
| `/autocontract` | Установить свой статус контракта |
| `/timelinetable` | Показать таблицу контрактов за период |
| `/help` | Показать список всех команд с навигацией |
| `/botinfo` | Информация о боте |

### Команды администратора

| Команда | Описание |
|---------|----------|
| `/settings` | Настройка бота (каналы, роли) |
| `/broadcast` | Отправить сообщение от имени бота |
| `/clean` | Очистка сообщений в канале |
| `/advertisement [канал]` | Создать объявление от администратора |
| `/setwelcomename` | Установить канал для приветствия |

### Команды разработчика (ID: 870408185620615212)

| Команда | Описание |
|---------|----------|
| `/devadvertisement` | Объявление от разработчика на все серверы |
| `/say [содержание]` | Сказать от лица бота |
| `/devlogbot` | Отправить логи консоли в чат |
| `/devcomandstatus` | Статистика использования команд |

## Функции

### Таблица контрактов
- Автоматическое создание таблицы в 00:00 по МСК
- Автоматическое закрытие таблицы в 23:59 по МСК
- Пользователи могут устанавливать статус: Онлайн, Форс-мажор, Оффлайн

### Приветствие новых пользователей
- Автоматическое приветствие в указанном канале
- Кнопка для изменения ника

### Логирование
- Логи изменения статусов
- Логи использования команд
- Консольные логи с перехватом

### Анимированный запуск
При запуске бот отправляет анимированное сообщение в канал логов:
```

Считывание данных бота...🤖
Загрузка системы [ ✅ ❌ ❌ ❌ ]
Загрузка системы [ ✅ ✅ ❌ ❌ ]
Загрузка системы [ ✅ ✅ ✅ ✅ ]
✅ Приложение успешно запущено
```
